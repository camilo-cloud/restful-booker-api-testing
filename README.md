# Restful Booker API Testing Portfolio

Manual QA testing project focused on validating the Booking module of the Restful Booker API.

## Objective

Demonstrate a complete API testing workflow by designing, executing, and documenting functional tests for the Restful Booker API using Postman and Newman.

## API Under Test

**Restful Booker** — a public API designed for practicing API testing. It provides authentication and full CRUD operations for hotel bookings.

- Base URL: `https://restful-booker.herokuapp.com`
- Documentation: `https://restful-booker.herokuapp.com/apidoc`

## Scope

This project covers the full Booking CRUD flow, authentication, health check, and both positive and negative test scenarios.

See [full scope definition](docs/scope.md) for details on what is included and excluded from this project.
